# my-survey-desktop

## Tauri

- Rust の実行環境

```bash
brew install rustup
```

- プロジェクト作成

```bash
# 新規作成
pnpm create tauri-app

# 開発用
pnpm tauri dev

# ビルド
pnpm tauri build
```

> https://tauri.app/v1/guides/getting-started/setup/
